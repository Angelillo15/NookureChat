# Nookure Chat

The ultimate chat plugin for your minecraft server, built for modern servers.

## Features

- Chat format by groups
- Words blacklist 
- Broadcasts
- Join & leaves messages
- URL block
- Anti Spam
- Clear chat
- Mentions
- Command block (in progress)
- Legacy support (in progress)
